9.2 Task 1：华灯物料编码规范设计

9.2.1 编码结构方案

根据华灯集团业务特点，你提出以下8位编码方案供客户审议：

位置位数含义取值范围/规则
第1-2位2位字母物料大类代码RM=原材料 / SF=半成品 / FG=成品 / PK=包装 / SV=服务
第3-4位2位数字产品线代码01=照明 / 02=小家电 / 03=配件 / 99=通用
第5-8位4位数字流水号（系统分配）0001起，按物料大类+产品线独立流水
编码示例：
RM01-0001 = 照明事业部原材料第1号
FG01-1520 = 照明事业部成品第1520号
SF02-0088 = 小家电事业部半成品第88号
配置位置（S/4HANA SPRO路径）：
SPRO → Master Data Governance → General Settings → Data Model → Define Entity Types
→ 找到MM（物料）数据模型 → Number Range → 按物料大类分段配置号码范围

📝 知识测验

Q1. 华灯集团MDG-M物料编码规则中，"FG02-0155"代表什么意思？
✅ 解析：FG=成品（Finished Goods），02=小家电产品线，0155=流水号。根据编码规则：RM=原材料，FG=成品，SF=半成品，PK=包装；01=照明，02=小家电。
Q2. BRF+规则MATCD-003（物料类型与编码前缀匹配）的作用是防止什么问题？
✅ 解析：如果ROH（原材料）的物料用了FG（成品）的编码前缀，会造成编码含义混乱，导致报表统计错误。BRF+强制校验类型与前缀的一致性。
Q3. 物料停用CR类型（HLD_MATDISCONT）中，"初审"步骤的主要目的是什么？
✅ 解析：物料停用前必须检查：①有无未结PO/SO；②库存是否为零；③有无正在使用的BOM引用。这些都是停用前置条件，必须在初审步骤由系统或管理员确认。
Q4. UAT测试场景MDG-M-TC002的目的是验证什么？
✅ 解析：TC002专门测试：利润中心为空时，BRF+是否能正确拦截并报错。这验证了S/4HANA最重要的数据质量规则之一。
Q5. 以下哪个SPRO路径用于配置MDG-M的Change Request类型？
✅ 解析：MDG的CR类型配置路径：SPRO → Master Data Governance → General Settings → Process Modeling → Change Requests → Define Change Request Types。这是MDG配置的标准入口。
